#### 2. Генерирует клиента Prisma на основе схемы 
```
npx prisma generate
```
--- 

#### 3. Синхронизирует состояние схемы Prisma с БД #### 
```
npx prisma db push
``` 
--- 

#### 4. Просмотр и управление данными через браузер
```
npx prisma studio
```
